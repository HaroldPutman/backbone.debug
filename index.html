a better way to debug your Backbone.js application

Backbone.debug is a small library that integrates with Backbone.js to easily provide common debugging functionality.

Simply include Backbone.debug.js directly after Backbone.js to get started!

Event Logging

Backbone.debug automatically tracks and logs all triggered events. Change events are grouped together per object for clarity.

Sync Logging

Calls to Backbone.sync are tracked and logged to provide better insight into the interactions that are occuring within your application.

Application Objects

Backbone.debug allows you to quickly see all instantiated collections, models, and views. The output will provide a quick overview of the state of the project.
