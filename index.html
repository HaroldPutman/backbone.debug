Backbone.debug

A small debug library for Backbone.js

Download Version 0.2.0

a better way to debug your Backbone.js application

Backbone.debug is a small library that integrates with Backbone.js to easily provide common debugging functionality.

Simply include Backbone.debug.js directly after Backbone.js to get started

Event Logging

Event logging is central to debugging event driven applications. Backbone.debug automatically tracks and logs all events.

The tracked events are intelligently grouped by object and event type. This grouping helps to organize the output logically and succiently. This functionality ensure that if you sync an object and a change event is fired for all attributes, only a single event will be logged containing all of the attributes that were changed

Sync Logging

Calls to Backbone.sync are tracked and logged to provide better insight into the interactions that are occuring within your application. These logs provide information about the object, the type of ajax request, and the url of the request.

Application Objects

Backbone.debug allows you to quickly see all instantiated collections, models, and views. The output will provide a quick overview of the state of the project.
